No copy, adaptation, port, translation, paraphrase, close rewrite, API-shape reuse, file-layout reuse, naming-pattern reuse, test reuse, example reuse, documentation text reuse, or recognizable implementation derivation is allowed unless separately reviewed, approved, licensed, and attributed.

| Source / License situation | Decision |
|---|---|
| MIT / BSD / Apache | Possible only after explicit license compatibility review, attribution plan, owner approval, and documented reuse decision. |
| GPL / LGPL | Do not import into the main codebase unless owner explicitly accepts the license implications in writing. |
| Unknown / no license | Research-only inspection; no reuse or derivative work. |
| Datasheets / official docs | Reference only; no long text/table copying. |
| Existing public examples/tests | Research-only unless separately approved and license-compatible. |

Official implementation source of truth:

- TI official datasheets/application notes

GitHub public availability does not imply permission to reuse code. Without an explicit license, default copyright rules apply.
