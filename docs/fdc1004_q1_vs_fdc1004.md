# FDC1004-Q1 vs FDC1004

FDC1004-Q1 is the automotive-qualified variant/documentation path. This library does not claim automotive qualification or automotive safety readiness.

FDC1004 and FDC1004-Q1 share the direct capacitance-to-digital measurement model relevant to the driver API. Q1 documentation should be used for automotive context and limits, not as proof that this Arduino library is automotive-grade.
